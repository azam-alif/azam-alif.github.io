How To Set Up a TFTP Server

Home Topics Contact

A TFTP server allows you to transfer files easily over your network. It is like a machine gun, where you can transfer data very fast when efficiency is not your concern. Follow these steps to set it up on Linux.

1. Install tftp-hpa

  • Arch Linux:
sudo pacman -S tftp-hpa

2. Create the TFTP Directory

Create a directory to store your files:

sudo mkdir -p /srv/tftp

Then put the files in that folder that you want to share with your tftp server.

Set Permissions

Change ownership and permissions:

sudo chown -R nobody:nobody /srv/tftp
sudo chmod -R 755 /srv/tftp

3. Configure TFTP

Edit the configuration file:

sudo vim /etc/conf.d/tftpd

Write there

TFTP_USERNAME="tftp"
TFTP_DIRECTORY="/srv/tftp"
TFTP_ADDRESS="0.0.0.0:69"
TFTP_OPTIONS="--secure"

4. Start the TFTP Service

Enable and start TFTP:

sudo systemctl enable tftpd
sudo systemctl start tftpd

5. Test the TFTP Server

On the client machine, you can test with the ip address of your server. But if you don't have any, then you can try on your own machine.

tftp localhost

This will open the TFTP CLI.

Congratulations!You have successfully created and enetered into the tftp server. Type quit to exit from tftp terminal.
